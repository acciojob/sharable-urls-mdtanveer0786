Shareable URL Generator

Generated URL:

https://localhost:8080/

How to use:

  1. Enter values in the name and/or year fields
  2. Click the "Generate URL" button
  3. The URL below will update with query parameters
  4. Copy and share the URL with others

Example Cases:

Both fields filled:

Name: "John"

Year: 2023

Result: https://localhost:8080/?name=John&year=2023

Only name filled:

Name: "Alice"

Year: (empty)

Result: https://localhost:8080/?name=Alice

Only year filled:

Name: (empty)

Year: 2024

Result: https://localhost:8080/?year=2024

Both fields empty:

Name: (empty)

Year: (empty)

Result: https://localhost:8080/
